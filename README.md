# A Vue2 Project Template with Vue2.7 & Vite

## Feature

☁️ A lightweight basic template with Vue2.7 & Vite

## Use it

```shell
npx degit doooooit/vue2.7-vite-template my-app
cd my-app
pnpm i # use `npm i -g pnpm` to install pnpm
```
